ID e CLASS

ID

O ID funciona como um CPF. Assim, como só posso ter um número de CPF, cada elemento só pode ter um valor de ID. O CPF é único, ou seja, ningúem pode ter um CPF igual ao seu. Isso significa que o ID também é único, então não é possível repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de manaeira oposta a ID. Diferente do ID em que você só pode ter um valot, no CLASS você pode ter vários valores. Também diferente do ID, em que não se pode repetir um valor, no CLASS você pode ultilizar o mesmo valor para vários elementos
